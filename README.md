# Login Bot 🔐

A beginner Selenium automation project that automatically logs into a website using Python.

---

## 🚀 Features

- Opens Chrome browser automatically
- Opens login webpage
- Enters username
- Enters password
- Clicks login button
- Uses Selenium waits for stable automation

---

## 🛠 Technologies Used

- Python
- Selenium
- ChromeDriver
